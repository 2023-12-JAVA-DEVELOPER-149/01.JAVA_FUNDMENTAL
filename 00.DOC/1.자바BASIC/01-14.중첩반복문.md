# 14. 반복문이 여러 번 포개진 중첩 반복문

## 중첩 반복문이란?

- 반복문 내부에 또 다른 반복문이 있음

- 여러 겹으로 반복문을 겹쳐서 구현 가능 (단 수행시간에 문제가 발생할 수 있음)

- 외부 반복문과 내부 반복문 간의 변수 값 변화에 유의 하며 구현하여야 함


##  for와 while의  중첩반복문을 사용해서 별찍기

		★★★★★
		★★★★★
		★★★★★
		★★★★★
		★★★★★

```java

public class NestedWhileForTest {

	public static void main(String[] args) {
		
		/*
		★[0,0]★[0,1]★[0,2]★[0,3]★[0,4]
		★[1,0]★[1,1]★[1,2]★[1,3]★[1,4]
		★[2,0]★[2,1]★[2,2]★[2,3]★[2,4]
		★[3,0]★[3,1]★[3,2]★[3,3]★[3,4]
		★[4,0]★[4,1]★[4,2]★[4,3]★[4,4]
		 */
		int i=0;
		while(i<5) {
			int j=0;
			while(j<5) {
				System.out.printf("%s[%d,%d]","★",i,j);
				j++;
			}
			System.out.print("\n");
			i++;
		}
		
		/*
		★[0,0]★[0,1]★[0,2]★[0,3]★[0,4]
		★[1,0]★[1,1]★[1,2]★[1,3]★[1,4]
		★[2,0]★[2,1]★[2,2]★[2,3]★[2,4]
		★[3,0]★[3,1]★[3,2]★[3,3]★[3,4]
		★[4,0]★[4,1]★[4,2]★[4,3]★[4,4]
		 */
		for(i=0;i<5;i++) {
			for(int j=0;j<5;j++) {
				System.out.printf("%s[%d,%d]","★",i,j);
			}
			System.out.print("\n");
		}
	}
}

```
