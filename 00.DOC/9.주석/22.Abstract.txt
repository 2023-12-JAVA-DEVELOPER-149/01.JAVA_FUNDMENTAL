  1.정의: 추상메쏘드가 존재하지 않으나 객체생성을 막기위해 만든 클래스
     ex> public abstract class Test{
         	//추상메쏘드();

         }

            ex> public abstract int print(int i);             
  2. 추상클래스는 불완전한 추상메쏘드를 가지므로 객체생성

      Test t=new Test();(X)

     구현(오버라이딩)하는 자식 클래스를 만들어 
     사용(자식객체생성)해야한다
